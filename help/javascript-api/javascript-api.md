# JavaScript-API

Client-seitige Marketo-JavaScript-Integrationen bieten Lead-Tracking, Forms, Web-Personalisierung und Funktionen für prädiktive Inhalte. Sie müssen über ein Marketo-Konto verfügen, um diese Funktionen nutzen zu können.

Die Implementierung umfasst normalerweise das Hinzufügen von Einbettungs-Code zu Ihrer Web-Eigenschaft. Sie können auch die JavaScript-Funktionen aufrufen, die vom Einbettungs-Code bereitgestellt werden, um Funktionen hinzuzufügen.

Der Einbettungs-Code ist für Ihre Marketo-Instanz eindeutig, da er eine Kontokennung enthält. Wechseln Sie in der Marketo-Benutzeroberfläche zum entsprechenden Bedienfeld, kopieren Sie den Einbettungs-Code in die Zwischenablage und fügen Sie ihn in Ihre Web-Seite ein.

## Lead-Tracking (Munchkin)

Der [Munchkin JavaScript-Trackingcode](lead-tracking.md) von Marketo generiert Leads von den Besuchen auf Ihrer Website. Es verfolgt auch Besucher, die keine personenbezogenen Daten angegeben haben, und erstellt anonyme Leads, die die IP-Adresse des Benutzers und andere Informationen enthalten.

Richten Sie Munchkin auf der Seite &quot;Munchkin&quot; im Admin-Bereich von Marketo ein.

## Formulare 2.0

Mit [Forms 2.0](forms-api-reference.md) können Marketing-Fachleute Web-Formulare erstellen, ohne Programmierkenntnisse zu haben. Forms kann sich auf Marketo-Landingpages befinden oder auf einer beliebigen Seite Ihrer Website eingebettet sein.

Verwenden Sie die Forms 2.0 JavaScript-API, um die Kernfunktionen eines Marketo-Web-Formulars zu erweitern.

## Web-Personalisierung

[Mit Marketo Web Personalization](web-personalization.md) können Sie Interessenten in Echtzeit auf Ihrer Website ansprechen, je nachdem, wer sie sind und was sie tun.

## Prädiktive Inhalte

[Marketo Predictive Content](predictive-content.md) nutzt maschinelles Lernen und Predictive Analytics, um Web-Besuchern relevante Inhalte zu präsentieren. Fügen Sie Ihrem Inhalt Textbeschreibungen und Bilder hinzu und betten Sie mehrere Inhaltsempfehlungen in Ihre Website ein.
